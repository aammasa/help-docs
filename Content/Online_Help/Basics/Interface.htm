User interface

The ProjectSight user interface consists of the following areas:

  1. Banner: Displays the global navigation elements.
  2. Navigation toolbar: Displays the navigation elements, page titles, and buttons related to the selected module.
  3. Project menu: Displays the modules that you can open.
  4. Content area: Displays the content associated with the selected module. You can create custom views and update information in the grids.
  5. Right-side panel: Displays information or actions related to the content area. For more information, see Panels.
Charlie assistant chatbot

Charlie is an AI chatbot where you can find information from the online help. By default, Charlie is available on the bottom left of the application, but you can reposition it if needed.

To reposition Charlie
  • Select and drag the chatbot icon or the top of the chat area to another position.

    • The chatbot remains in that position as you navigate around the application.

To restart your conversation
  1. At the top of the chatbot window, select (More) > Restart conversation.
  2. In the chat area, select Yes.

    3. Your conversation is restarted, and the welcome message is displayed again.

To activate the chatbot
  1. On the banner, select (Settings).
  2. In the settings menu, select My settings > AI Assistant.
  3. Select the Activate Charlie checkbox.
To deactivate the chatbot
  1. At the top of the chatbot window, select (More) > Deactivate Charlie.
  2. If the confirmation dialog is displayed, select Deactivate.
    3. Tip

    You can also deactivate the chatbot in the settings (). In the settings menu, select My settings > AI Assistant, and then clear the Activate Charlie checkbox.

Company Search dialog

In the Company Search dialog, which is available from company fields such as From Company, you can search for companies or create them.

To open the Company Search dialog
  • In a company field, select (Search).
To search for a company
  1. Open the Company Search dialog.
  2. On the Select tab, turn on the Project Companies toggle.

    3. A project company has at least one contact with access to the current project.

    Example

    Company contact with access to the project

  3. To filter the list by a column, enter a value in the filter row.
  4. Select the company name, and then select Select.
To create a company
  1. In the Company Search dialog. select the Create tab.
  2. In the Name field, enter a unique company name.
  3. Enter the company details.
  4. Select Create.

    5. A company record is created in the company settings.

Contact cards

Contact cards represent people who are associated with the project, record, or file. These cards are displayed in places such as the Information panel or Contact Info tab of records.

To view more information about a contact
  • On the contact card, hover over the profile picture or initials.

    • The contact information comes from the Contacts tab of the company record.

To copy the contact details
  1. On the contact card, hover over the profile picture or initials.
  2. In the contact popover, select the contact information that you want to copy.

    3. The contact information is copied to your clipboard.

To see other available actions
  • On the contact card, select (More).

    • The available actions depend on where the contact card appears.

Dialogs

You can move and resize various dialogs such as child record, search, and print dialogs.

To move a dialog
  • Select the top area of the dialog, and then drag it.
To resize a dialog
  • Select the side or corner of the dialog, and then drag it.

    • —Or—

    To change the dialog to the maximum size, select (Maximize).

    —Or—

    To change the dialog from the maximum size to the previous size, select (Restore).

Grids

Some areas contain grids where you can add and update information. You can also edit the grid columns.

To edit a cell
  • Select the cell, and then edit the information.
    • Tip

    You can press Tab to move to the next cell in the row.

To copy text from a cell
  • Right-click the cell, and then select Copy Text.
To copy values down a column
  1. Select an unlocked cell that contains the value to be copied.

    2. The cell will either be selected or the dropdown list will be expanded.

  2. Press Ctrl+Down arrow.

    3. The value is copied down to the unlocked cells in the column.

    This keyboard shortcut works for columns where you directly edit or select a value. It doesn't work for columns, such as Location, where you select a value from a popup or in grids that are saved automatically.

To delete a row
  • Select the checkbox for one or more rows, and then select Remove.

    • The rows are deleted when you save the changes.

Views

In the content area, you can set up the data view according to your preferences. If you save the changes as a custom view, you can look at the information again with the same settings.

To switch between the tile and list views
  • At the top-right of the view, select (Tile) or (List).
To edit the columns in the list view
  1. At the top-right of the view, select (Columns).

    2. —Or—

    On the right-side of the grid header row, select (Columns).

    Tip

    You may need to scroll to the right to see the button.

  2. Select or clear the checkboxes for the columns.
  3. Select Apply.

    4. The selected columns are displayed in the grid.

  4. To adjust the width of a column, place the pointer between two columns, and then select and drag the border.
    5. Tip

    To automatically resize the column for the content, double-click the border.

  5. To reorder the columns, select and drag a column header to a new position in the grid.
To sort by columns in the list view
  1. In the list view, select a column header.

    2. The items are sorted in ascending order.

  2. To sort the items in descending order, select the same column header.

When applicable, the columns are sorted by the selected column, then by the Number column, and then by the Revision column.

To group the items
  • At the top-right of the view, select the Group list, and select an option.

    • Within each group, the items are sorted by the number in ascending order.

    Tip

    To quickly collapse or expand the groups, you can select (More) > Collapse All or Expand All on the navigation toolbar.

To sort the items
  • At the top-right of the view, select the Sort list, and select an option.

    • Items are sorted alphanumerically in ascending order by the selected option and then by the number.

    Drawings can be sorted alphabetically or ‘intelligently’.

To refresh a view
  • On the navigation toolbar, select (View Options) > Refresh.

    • The view is updated, so you have the latest information from other users.

To export the current data view to Excel
  1. To modify the current view:
    • In the Search panel, change the search filters.
    • From the Group list, select an option.
    • From the Sort list, select an option.
    • In the list view, sort the items by column.
  2. Edit the columns in the view.
  3. On the navigation toolbar, select (View Options) > View in Excel.

    4. The Excel file includes all the visible columns and rows.
