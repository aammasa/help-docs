Custom views

Set up custom views so you can see the relevant information at a glance.

If custom views include project-specific data, they won't work across projects.

You can create a custom view or save the current view when you change any of the following:

  • Search filters
  • Grouping option
  • Sort order
  • Columns (including order and width)

You can also add custom views as dashboard widgets for drawing sets, drawings, and most record types. If this feature is available for the view, the widget checkbox is displayed in the View Settings dialog.

Example

Dashboard widget checkbox

To open a view
  • In the View list, select the view.
Creating custom views

You can save the settings in the current view as a new custom view.

To create a custom view
  1. To modify the current view:
    • In the Search panel, change the search filters.
    • From the Group list, select an option.
    • From the Sort list, select an option.
    • In the list view, sort the items by column.
  2. Edit the columns in the view.
  3. Open the View list, and then select (Add).
  4. Enter a unique name for the new view, and then select Save.
  5. Share the custom view.
Sharing custom views

You can share your custom views with selected users or security roles. The shared view displays the name of the person who created it.

To share a view
  1. In the View list, select (Settings) by the custom data view.
  2. In the Permissions grid, select (Add User).

    3. —Or—

    Select (Add Role).

  3. Select the checkbox for the users or roles, and then select Select.
  4. In the Permission column, select the permission for each user or role.
    • Edit: Edit the permissions, search criteria, columns, grouping, and sorting of the shared custom view and add it as a personal dashboard widget. The user can save changes to the shared custom view.
    • View: View the shared custom view and add it as a personal dashboard widget. If changes are made to the shared custom view, the user can save it as a new custom view.
  5. Select Save.
To remove access to a view
  1. In the View list, select (Settings) by the custom data view.
  2. In the Search field, enter the name of the user or role, and select a search filter.
  3. In the Permissions list, select the checkbox for the users or roles that will be removed.

    4. —Or—

    To remove your own access to a view that was created by another user, select Delete.

  4. Select (Remove), and then select Delete.
  5. Select Save.
Editing views

You can edit the default view or a custom view.

If you rename a workflow status after it's included in a custom view, the custom view isn't updated automatically for the renamed status. You can edit the custom view to include the renamed workflow status.
Example
  1. You create a custom view called New RFIs that shows RFIs with a workflow status called New.
  2. When you apply the New RFIs view, the RFIs with the New status are displayed.
  3. You change the name of the workflow status from New to Review.
  4. When you apply the New RFIs view, no RFIs with the Review status are displayed.
  5. In the Search panel, you turn on the Review toggle.
  6. In the View list, you save the changes to the New RFIs view.
  7. When you apply the New RFIs view again, the RFIs with the Review status are displayed.
To edit a view
  1. In the View list, select the view.
  2. To modify the current view:
    • In the Search panel, change the search filters.
    • From the Group list, select an option.
    • From the Sort list, select an option.
    • In the list view, sort the items by column.
  3. Edit the columns in the view.
  4. Open the View list, and then select (Save).
To expand groups by default in the custom view
  1. In the View list, select (Settings) by the custom data view.
  2. Select the Expand Groups by Default checkbox.
  3. Select Save.
To rename a custom view
  1. In the View list, select (Settings) by the custom data view.
  2. In the View Name field, edit the name.

    3. You can edit the name if you created the custom view.

  3. Select Save.
To reset the default view
  1. In the View list, select (Reset) by the default view.
  2. In the list, select Yes.

    3. The standard settings are restored for the default view.

Deleting custom views

You can delete custom views but not the Default View.

Shared views can only be deleted by the user who created them.

To delete a custom view
  1. In the View list, select (Settings) by the custom data view.
  2. In the View Settings dialog, select Delete.
