Panels

You can view information and perform different actions in the right-side panels. The most recently used panel for the item type is open by default.

To open or close a panel
  • On the navigation toolbar, select the button.
To resize the Create, Information, or Search panel
  • Select and drag the edge of the panel.

    • —Or—

    Double-select the edge of the panel to reset it to the default width.

(Assignments)

The Assignments panel displays the records, drawings, and specifications with open assignments that are assigned to project contacts. Each item is displayed only once, even if it has multiple assignments, and shows the earliest due date.

  • The panel only displays the unlocked items that you have access to.
  • If an assignment doesn't have a due date, the record due date, if available, is used.

When you open an item from the Assignments panel and update the assignment, the panel is updated only for the item that's opened directly from the panel.

Example
  1. From the Assignments panel, you open Item A and update the assignment.
  2. From Item A, you open Item B and update the assignment.
  3. When you close the items, the Assignments panel is only updated for Item A.
(Columns)

The Columns panel is available in various list views. You can select the columns that are displayed and in what order.

(Create)

The Create panel is available for collaborative records. You can create records and add assignments, courtesy copies, and comments to them without leaving the selected view.

  • If the right-side panel is open when you select the button, the Create panel opens.
  • If the right-side panel is closed when you select the button, an empty draft record opens in the full form.
(Information)

In the Information panel, you can view or edit the details of a record, drawing, or file.

For collaborative records, you can add assignments, courtesy copies, and comments in the panel without leaving the selected view.

  • If the right-side panel is open when you select a record, the record details appear in the Information panel.
  • If the right-side panel is closed when you select a record, the full record opens.

You can also edit collaborative records in the Batch Edit panel.

Tip

Changes are saved automatically in the Information panel when you:

  • Close the panel or select a different panel.
  • Select another tile or row in the data view.
  • Navigate away from the current data view.

If you select (Expand) before saving the changes, they're displayed in the record but aren't saved yet.

(Layers)

The Layers panel, where you can filter annotations, is available when you open a published drawing, a specification set, or published specification.

(Links)

In the Links panel, you can view and add linked items for a record or drawing. The Links panel only displays items that you have access to.

The linked items are organized on the following tabs:

  • Files: Linked files (excluding drawings).
  • Records: Linked records and specification sets.
  • Designs: Linked drawings and specifications.
  • Web: Hyperlinks (URLs).

Depending on the item, the link is one-way (only appears in the originating item) or two-way (appears in the originating item and the linked item).

(Search)

In the Search panel, you can filter the items in the current view. The filters remain applied (but the search results aren't refreshed) if you:

  • Close the Search panel.
  • Open an item from the results and then return to the view.

When you go to a different view, the filters aren't retained.

After you delete a lookup list item, it remains available in the Search panel filter if it's part of a saved custom view or if it's selected for a record.

To set the search filters
  1. To open the Search panel, select (Search) on the navigation toolbar.
  2. In the Search field, enter the search terms.

    3. In the list view, the search terms are highlighted in the matching items.

  3. Set the other search filters, which vary depending on the view.
    • To filter the results by a category, turn on the category toggle.
    • To set a date range, select (Date), and select the dates.
      • Tip

      To add the current date in a date field, select the field, and then press Ctrl+D.

    • To filter the results by linked specifications, select the Linked Specifications field, and then select one or more specifications.
      • Example

      Linked Specifications field in the Search panel

    • To filter the results by a field, select the field in the By section. You can select multiple fields.

      • All the options in the available items are displayed.

To clear the search filters
  • In the Search panel, select Reset Search.
(Sharing)

In the Sharing panel, you can view and modify who has access to a record. You can also select a different author in the Author section of the panel.
