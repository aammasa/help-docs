Dashboards

The project dashboard, which is personal to each user, displays widgets with information from the items that you have access to.

To open the dashboard
  • In the project menu, select Home.
System dashboard widgets

Several system dashboard widgets are available. You can also add custom view widgets based on your custom views.

Assigned items

The following widgets are for assigned items:

  • Overdue Items: The assigned items, grouped by type, that are overdue.
  • Due Today Items: The assigned items, grouped by type, that are due today.
  • Due in Next 7 Days: The assigned items, grouped by type, that are due in the next 7 days.

The information in these widgets come from the Assignments panel.

Records and drawings

The following widgets are for records and drawings:

  • Records - Various types - The record that have the five selected workflow statuses. Some widgets display the average turnaround, which is the average number of days from the Created On date to the resolution or answer date.
  • Drawings - The drawings that have the selected workflow statuses. Only the Current and Not Current statuses are available for dashboard widgets.

Projects

The Work Progress widget is based on the start and end dates from the Project form. The progress is displayed if the start date is today or in the past.

Custom view dashboard widgets

You can add your custom views as dashboard widgets.

  • You can add a shared custom view as a widget on your dashboard. If your access to the custom view is removed, the custom view widget is also removed from your dashboard.
    • Example

    Add a custom view as a dashboard widget

  • Custom view widgets are removed automatically from the dashboard if you save them without any search filters.
    • Example

    In the RFI module:

    1. You apply the Assigned to Me search filter and group the RFIs by due date.
    2. You save the current view as a custom view called Due RFIs.
    3. You add the Due RFIs view to the dashboard.
    4. Later you remove the search filter but keep the grouping and save the Due RFIs view.

      5. The Due RFIs view no longer appears on the dashboard or in the dashboard organizer.

To add a custom view widget to the dashboard
  1. Create a custom view with at least one filter applied from the Search panel.
  2. In the View list, select (Settings) by the custom data view.
  3. In the View Settings dialog, select the Include as widget on my dashboard checkbox.

    4. The custom view is added to the dashboard organizer and at the bottom of the dashboard.

  4. In the dashboard organizer, reorder the dashboard widgets.
To remove a custom view widget from the dashboard
  1. In the View list, select (Settings) by the custom data view.
  2. In the View Settings dialog, clear the Include as widget on my dashboard checkbox.

    3. The custom view is removed from the dashboard and the dashboard organizer.

    Tip

    To keep the custom view widget in the dashboard organizer but not display it on the dashboard, you can select (Remove) on the widget.

Organizing dashboard widgets

In the dashboard organizer, you can add, reorder, and remove dashboard widgets.

Default dashboard widgets

The following dashboard widgets are displayed in the dashboard by default:

  • Overdue Items
  • Due Today Items
  • Due in Next 7 Days
  • Punch Items
  • RFIs
  • Submittals
  • Work Progress
  • Custom views
To open the dashboard organizer
  • On the toolbar of the dashboard, select (Dashboard organizer).
To organize the dashboard widgets
  1. Open the dashboard organizer.
  2. To add widgets to the dashboard, select and drag widgets from the Available Widgets section to the My Dashboard section.
    3. Tip

    To quickly find a widget in the Available Widgets section, select a filter from the list.

  3. To remove widgets from the dashboard, select and drag widgets from the My Dashboard section to the Available Widgets section.
  4. In the My Dashboard section, select and drag the widgets into the desired order.

    5. The widgets are displayed in order from left to right and from top to bottom. Your screen size determines the number of widgets per row.

  5. Select Save.
To display all the default and custom view widgets on the dashboard
  1. In the dashboard organizer, select Reset.

    2. All the default and custom view widgets are moved to the My Dashboard section of the dashboard organizer.

  2. Select and drag the widgets into the desired order.
To remove a widget from the dashboard
  • On the widget, select (Remove).

    • In the dashboard organizer, the widget is moved from the My Dashboard section to the Available Widgets section.

Working with dashboard widgets

In the widgets, you can see more details or open a filtered list of items.

Tip

To see the latest information in the dashboard, you can refresh the page.

To see more details in a widget
  • Point your cursor to a place on the graph or chart.

    • The tooltip displays information such as the type and number of records.

To filter the due items
  • On the assigned items widget, select (Edit), and then select an option.
    • Tip

    If multiple record types are tied in the top three or top five, each one is displayed separately in the widget.

To view a list of due items by type
  1. On the assigned items widget, select a section.
  2. To open an item, select the item in the list.
To view a filtered list of records
  • On the widget, select the number under the workflow status.

    • If a filtered list is unavailable for a status, the number isn't clickable.

To open a record view from a widget
  • Select the widget title.
